My-To-Do Manager

This web application was built with the purpose to make the classic My-To-Do Lists something easier to do! You can manage all of your tasks and duties records (them, add, update, remove and even save them).

In future versions we hope to add integration with already existing reminders applications, such as Google Calendar or iOS reminders native applications. However, use API's to build communications from this SPA it's absolutely necessary, so incorporate a backend service will be important too.

This software architecture is based on a three layered application concept: data (classes designed to manage information), logic (all the algorithms to support the entire SPA functionality) and interface (graphical interfaces for user interaction).

Software developed with Angular v14 in march, 2023.
